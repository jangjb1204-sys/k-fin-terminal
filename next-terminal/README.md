# K-Fin Terminal Next

Next.js/React version of the Korean financial terminal.

## Local Run

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Deploy

Use Vercel.

- Root directory: `next-terminal`
- Build command: `npm run build`
- Output: Next.js default

This version keeps the Streamlit app intact and can be deployed separately.
